Consultar o CPF
Para consultar o CPF na API, você precisará seguir os passos abaixo:

1. Obtenha um token de acesso
Para consultar o CPF na API, você precisará de um token de acesso. Você pode obter um token de acesso criando uma conta no nosso site.

2. Faça uma requisição GET
Para consultar o CPF na API, você precisará fazer uma requisição GET para o endpoint /consulta/{token}/{cpf}.

3. Insira o token de acesso e o CPF
Insira o token de acesso e o CPF que você deseja consultar no endpoint.

4. Receba a resposta
A API retornará a resposta com as informações do CPF consultado.
